## 构建环境

- **Gradle**: 8.13
- **Android Gradle Plugin**: 8.13.2
- **Kotlin**: 1.9.22
- **compileSdkVersion**: 33
- **Java**: 21

### 国内镜像配置

项目已配置国内镜像加速，包括：
- 阿里云镜像 (google, central, public, gradle-plugin)
- 华为云镜像
- 腾讯云镜像
- Gradle 下载源使用腾讯云镜像
